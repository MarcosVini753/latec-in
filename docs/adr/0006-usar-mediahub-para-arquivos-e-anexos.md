# ADR 0006: Usar mediahub para arquivos e anexos

## Status
Proposta

## Contexto
Projetos, notícias, cursos e materiais podem compartilhar imagens, PDFs, e-books, livros e documentos técnicos.

## Decisão
Criar um app `mediahub` para centralizar ativos de mídia e documentos reutilizáveis.

## Consequências
Evita duplicação de uploads e facilita reaproveitamento de arquivos em múltiplas áreas do portal.
