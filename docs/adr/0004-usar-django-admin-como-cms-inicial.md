# ADR 0004: Usar Django Admin como CMS inicial

## Status
Aceita

## Contexto
A LATEC.IN precisa gerenciar conteúdos sem construir imediatamente um painel administrativo customizado.

## Decisão
Usar Django Admin como CMS inicial para membros, projetos, notícias, cursos, materiais, parceiros e mensagens.

## Consequências
A equipe reduz tempo de desenvolvimento inicial. Um painel próprio poderá ser avaliado depois, caso o Django Admin se torne insuficiente.
